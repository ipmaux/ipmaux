How to Apply Different Number Formats to Excel Columns

Before applying specific commands in an Excel spreadsheet, data in different cells within a singular column must all have the same number formatting.

An example of a spreadsheet in which cells within the same column have inconsistent number formatting
Student ID Academic Program Action Date Fees
010312521 ACC 1/15/2021 1299
103671192 ACF Thursday, January 14, 2021 1399.99
103680213 BAB 12/15/2020 1209.99
103726214 BBS Friday, December 11, 2020 1209

Applying the Short Date Number Format

  1. Select the entire column or cell(s) that require formatting. In this example, column E (Action Date) should be expressed in the same date format, which will be the short date option.
    Select column E by moving the cursor over the column E label. The cursor should display a black downward arrow and select it. All of column E should be highlighted, indicating that the entire column and its data are selected.
  2. Select the short date option from the number format drop down menu withn the Number group command.

Applying a Currency Number Format

The final column that requires consistent number formating is column H (fees). A dollar value format consistent to two decimal places is required.
To select all of column H's contents, follow the step 1 process from the above Applying the Short Date Number Format example.

  1. Once column H is selected, select the currency numbert format option from the number format drop down menu within the Number group command.
  2. To assign a consistent decimal format use the increase or decrease decimal commands within the Number group command until all values in the column are expressed to two decimal places. The number values in this column are now all expressed as currency amounts to two decimal places.

Now that there is consistent number formatting within all columns, more advanced formatting, filtering, and sorting can be applied to the dataset.
